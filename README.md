# HR Data Analysis with Python

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on an HR dataset using Python. It analyzes employee information such as department, salary, experience, work mode, performance ratings, and employee status to generate meaningful business insights through data visualization.

## 🚀 Features
- Data cleaning and preprocessing
- Date format conversion
- Employee status analysis
- Department-wise employee distribution
- Job title analysis
- Average salary analysis by department and job role
- Performance rating analysis
- Experience-based insights
- Interactive charts using Matplotlib and Seaborn

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Analysis Performed
- Employee status distribution (Active, Resigned, Terminated, etc.)
- Work mode distribution
- Department-wise employee count
- Job title frequency
- Average salary by department
- Average salary by job title
- Department vs Job Title salary comparison
- Resigned and terminated employee analysis
- Performance rating by department
- Experience vs salary analysis

## 📂 Dataset
The project uses an HR dataset (`HR_Data.csv`) containing employee details such as:
- Department
- Job Title
- Salary
- Experience
- Hire Date
- Work Mode
- Performance Rating
- Employee Status
- Location

## ▶️ How to Run
1. Clone the repository.
2. Install the required libraries:
   ```bash
   pip install pandas matplotlib seaborn
   ```
3. Place `HR_Data.csv` in the project directory.
4. Open and run `hr-data-analysis-with-python.ipynb`.

## 📈 Outcome
The project provides valuable HR insights that help understand workforce distribution, salary trends, employee performance, and attrition patterns using simple and effective visualizations.
